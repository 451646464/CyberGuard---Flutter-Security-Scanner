# 🔒 Secyrity - تطبيق تقييم أمان Android

تطبيق Flutter احترافي لتقييم أمان هاتف Android يعتمد على بيانات حقيقية من الجهاز.

## المميزات

- 🔒 تقييم درجة أمان الجهاز
- ⚠️ كشف التطبيقات الخطيرة
- 📡 فحص حالة الشبكة
- 🔔 تنبيهات أمنية فورية
- 📊 تقارير أسبوعية وشهرية

## المتطلبات

- Flutter SDK >= 3.0.0
- Android SDK >= 21 (Android 5.0+)

## التثبيت

```bash
flutter pub get
flutter run
```

## الصلاحيات المطلوبة

- `QUERY_ALL_PACKAGES` - لفحص التطبيقات المثبتة
- `INTERNET` - لفحص حالة الشبكة
- `ACCESS_NETWORK_STATE` - لفحص حالة الاتصال

## ملاحظات

- جميع العمليات تتم محلياً بدون اتصال خارجي
- لا يتم إرسال أي بيانات إلى سيرفرات خارجية
- التخزين محلي باستخدام SQLite
